Need Advanced Configuration?

The wizard provides guided setup for common healthcare scenarios. For advanced users who need full control, an "Escape to Manual Configuration" option is available at any step.

Manual configuration includes:

Content Options:
  • Dynamic/Static data sources
  • Custom workflow & API integrations
  • Calendar column sizing (Fit/Count/Fixed)
  • View modes (Day/Week/Month)
  • Time format (12H/24H) & scale (1H/30MIN)
  • Working hours configuration
  • Actions (Add/Edit/Delete/View)
Style Customization:
  • Padding & spacing controls
  • Color roles (fills, outlines, accents)
  • Date selector states (default/hover/selected)
  • Appointment containers & labels
  • Grid lines & backgrounds
  • Font families, weights, sizes
Responsive Design:
  • Desktop/Tablet/Mobile visibility
  • Width & height sizing options
  • Conditional visibility rules
  • Custom themes (Clarity/Meridian/Pulse/Tonal)
  • Tag & chart color palettes
  • Border effects & corner radius

üí° Best Practice: Complete the wizard first to establish a solid foundation, then fine-tune with manual configuration if needed. The wizard prevents common timezone and configuration errors that are easy to make in manual mode.
